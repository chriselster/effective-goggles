Information We Collect

We collect the following types of encrypted user data from individuals who use the Doro app:

  2. Purchase History: We collect information about in-app purchases made by users for transactional and analytical purposes.
  3. Approximate Location: We collect approximate location information to enhance user experience, such as providing location-specific content and features.
  4. Diagnostic Data: We may collect diagnostic information about the performance of the app on a user's device to identify and resolve technical issues.
  5. App Interactions: We gather data about how users interact with the app, including features accessed, buttons clicked, and preferences selected.
  6. Device Identifiers: We collect unique identifiers associated with users' devices for purposes like app security and user support.

We use the collected information to:

  • Personalize user experiences and improve app features.

  • Ensure app security and protect user accounts.
